Design Guideline: Footer Wordmark

Usage

The gradient wordmark SVG (powerhouse-wordmark-gradient.svg) is used as a large-scale background element at the very bottom of the page. It spells "POWERHOUSE" in outlined letters with a holographic gradient stroke — subtle gray outlines accented by iridescent purple-to-blue light streaks at select letter junctions.

Implementation

The wordmark spans the full viewport width and is partially cropped so only the top portion of the letters is visible, creating a dramatic ground-level branding element.

  • Place outside any padded container — directly on the body so it spans the full viewport width
  • Container: overflow: hidden with a fixed height (~130px)
  • Image position: top: 0 — the container height controls how much of the letters is visible
  • Width: 110% with left: -5% — slightly oversized to push the first letter (P) off-screen, matching the reference
  • Crop level: the bottom edge of the container should sit just above the horizontal crossbar of the H — only the tops of the letters are visible
  • No opacity reduction needed — the SVG stroke is already rendered at low contrast against dark backgrounds
  • Works exclusively on dark backgrounds (#0B0D0F or similar)

Do not

  • Use on light backgrounds — the gradient stroke is designed for dark surfaces
  • Add additional opacity — the SVG's built-in transparency is calibrated
  • Scale below 100% viewport width — the text should always feel oversized and immersive
